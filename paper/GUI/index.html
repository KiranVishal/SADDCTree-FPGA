Automatic Generation of Hardware Tree Classifiers






Four simple steps to implement machine learning algorithms on an FPGA

  1. Select a machine learning algorithm to work with from the navigation bar.
  2. Upload a dataset for training the machine learning model.
  3. Select the different parameters desired for your design and click train.
  4. Compile the Chisel code to obtain the FPGA implementable hardware descriptive code.
